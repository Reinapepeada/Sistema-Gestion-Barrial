# Sistema de Gestión Barrial

Este es un proyecto de sistema de gestión barrial desarrollado con Flask.

## Configuración del entorno

Para configurar el entorno de desarrollo, sigue estos pasos:

1. Asegúrate de estar en la carpeta raíz del proyecto.

2. Abre la terminal en esta ubicación.

3. Ejecuta el siguiente comando para crear un entorno virtual:

    py -m venv .venv

4. Activa el entorno virtual ejecutando el siguiente comando:

    .venv/scripts/activate

5. Asegurar que estas dentro del directorio flask-barrial

    cd C:\Sistema-Gestion-Barrial\flask-barrial
 
6. Instala las dependencias del proyecto ejecutando el siguiente comando:

    pip install -r requirements.txt

## Ejecución del proyecto

Una vez que hayas configurado el entorno, puedes ejecutar el proyecto Flask. Para hacerlo, ejecuta el siguiente comando:
py run.py


# Para configurar la base de datos

1. Habilitar en el sql server el logueo por sql authentication
2. correr el script para crear tablas y datos
3. crear cuenta con usuario y contraseña en el archivo config.py(esta dentro de la url)
